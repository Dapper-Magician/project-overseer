# Estimate Calibration Log

This file tracks predicted implementation time versus actual implementation time.

The goal is not to defend estimates. The goal is to improve roadmap accuracy over time.

## Calibration Rules

For each implementation task, record:

- task name
- task description
- predicted time
- actual time
- tool or agent used
- prompt quality
- blockers
- outcome
- lessons learned
- future estimate adjustment
