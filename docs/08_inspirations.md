# Inspirations

## Purpose

This document records systems, interfaces, and workflows that inspire Project Overseer.

The goal is not to copy any one system.

The goal is to extract useful patterns and adapt them into the Project Overseer command institution.

## Inspiration Categories

### Agentic Coding Systems

Useful patterns:

- codebase-aware implementation
- task prompts
- file edits
- test runs
- review loops
- repo-aware development

Relevant inspiration:

- OpenAI Codex
- Claude Code
- Gemini CLI
- Cline
- Roo Code

Project Overseer adaptation:

- Overseer generates implementation tasks.
- Bolt refines technical specs.
- Darwin defines acceptance criteria.
- Codex or another coding agent implements.
- The team reviews and records results.

### Knowledge Workstations

## Design Rule

Borrow patterns.

Do not copy surfaces.

Project Overseer should feel original.
